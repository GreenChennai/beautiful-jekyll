### 上传的文件命名与MarkDown的写法

* 命名有如下规则
* 需要三个短横线   该不是中文字符> --- <是英文字符
* 分别填写年月日最后填写文件名字,比如> 2018-08-10-use English
* 年-月-日-文件名字
* 需要注意的是如果是单数比如2,不是单单写2而是要在前面加个0 变成> 02
* MarkDows语言的写法可以[看这里](https://www.cxanimeclub.ml/2018-10-01-MarkDown)
